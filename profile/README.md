## What we are building

EnsinoLibre is a free, open-source worksheet builder and a library of reusable teaching components. Teachers build, customize, and share worksheets from blocks, keep what works, and remix the rest. The name fuses *ensino*, the Portuguese word for teaching, with *libre*, the word open source uses for freedom.

We started with Iberian roots and we are built for classrooms everywhere. The software is free to use, free to change, and owned by the community that teaches with it.

## What we believe

- **Teachers come first.** The product saves time, so every screen earns its place. You should not need to be technical to build a great worksheet.
- **Open by default.** Code, components, and shared worksheets are transparent and remixable, always with credit.
- **Built together.** A typo fix, a translation, a classroom-tested worksheet, and a new feature are all real contributions.

## Where to start

| Repository | What it holds |
|------------|---------------|
| [`core`](https://github.com/ensinolibre/core) | The worksheet builder application |
| [`blocks`](https://github.com/ensinolibre/blocks) | Reusable teaching components |
| [`design-system`](https://github.com/ensinolibre/design-system) | Brand, tokens, and UI foundations |
| [`docs`](https://github.com/ensinolibre/docs) | Guides for teachers and contributors |
| [`website`](https://github.com/ensinolibre/website) | The public site at ensinolibre.org |
| [`translations`](https://github.com/ensinolibre/translations) | Locale files and the translator workflow |
| [`examples`](https://github.com/ensinolibre/examples) | Community worksheets to fork and adapt |

## Ways to help

- **Teachers.** Build a worksheet, share it to [`examples`](https://github.com/ensinolibre/examples), or tell us what slows you down.
- **Translators.** Add or improve a language in [`translations`](https://github.com/ensinolibre/translations). Portuguese and Spanish are first-class, and every other language is welcome.
- **Developers.** Pick up a [good first issue](https://github.com/ensinolibre/core/labels/good%20first%20issue) and read the [contributing guide](https://github.com/ensinolibre/.github/blob/main/CONTRIBUTING.md).
